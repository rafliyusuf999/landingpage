  • 02 Software programmer

    Software programmer merancang dan membuat perangkat lunak untuk berbagai sistem komputer. Setiap jenis program memerlukan bahasa pemrograman yang unik sehingga banyak programmer berspesialisasi dalam bahasa codingan tertentu.

  • 03 Database programmer

    Database Programmer bertanggung jawab atas desain, pemrograman, konstruksi, dan implementasi basis data baru, serta memodifikasi basis data yang ada untuk pembaruan platform dan perubahan kebutuhan pengguna

  • 04Web developer

    Web Developer adalah perancang sebuah website / www.
